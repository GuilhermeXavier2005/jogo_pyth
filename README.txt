----------objetivos do projeto

	O projeto inicialmente visa revisar e aprofundar os conceitos aprendidos em Fundamentos de Algoritmos e progredir em desenvolvimento de habilidades computacionais que a linguagem python pode prover.

o projeto deve contem um menu, dificuldades, variedades de palavras, aleatoriedade em seleção de palavras e dicas para o jogador.

-----------tecnologia usada:
	python versão 3.11
	editor Visual Studio Code

-----------como jogar:

modo médio -->
    há um indicativo de quantas letras tem na palavra, se o jogador acertar a posição e qual letra na sua 'tentativa' então a letra será exibida e ajudará a formar a palavra para advinhar
    quantidade de letras e chances são exibidas na tela pra ajudar o jogador
modo difícil --> 
    acréscimo de uma palavra para descobrir, aproveitando da mesma mecânica da dificuldade anterior

---------Como aplicar:

	1. Baixe os arquivos de extensão "py" e certifique que estão no mesmo diretório.
	2. Abra o Visual Studio Code, caso não tenha, instale em https://code.visualstudio.com/ e faça o download para extensão Python em extensões na lateral esquerda do programa. 
	3. Abra o terminal (ctrl + tecla ") e digite "python principal.py"
